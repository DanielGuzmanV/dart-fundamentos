# Para crear un proyecto similar:
````
* Presionamos las teclas de: CTRL + SHIFT + P

* Luego en la barra de busqueda que nos saldra pondremos:
  Dart: New Project (donde lo seleccionamos)

* Donde saldra unas opciones, pero se seleccionara:
  Console Application "A command-line application"

* Y finalmente se ira creando el proyecto, donde se trabajar en las carpetas de "lib" o "bin"

* Donde si se trabaja en la carpeta de "lib" comentar o eliminar el codigo de las otras carpetas, como ser las carpetas de:
"bin" y "test"

* Eso seria todo, FIN.
